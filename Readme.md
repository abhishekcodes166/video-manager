Video Manager 🎥

A full-stack MERN video sharing platform inspired by YouTube, where users can upload, manage, and stream videos seamlessly. The platform provides a modern and responsive UI along with powerful backend functionality for handling video content, user authentication, and engagement features.

🚀 Features
🔐 User Authentication & Authorization (JWT)
📤 Upload and manage videos
🎬 Stream videos smoothly
👍 Like & dislike videos
💬 Comment system
📂 Video categories and playlists
🔍 Search videos by title or category
📜 Watch history support
📱 Fully responsive design
☁️ Cloud storage integration for media files
⚡ RESTful APIs with secure backend architecture
🛠️ Tech Stack
Frontend
React.js
Redux / Context API
Tailwind CSS / CSS
Backend
Node.js
Express.js
MongoDB
Mongoose
Other Tools
JWT Authentication
Multer / Cloudinary
Axios
📌 Purpose

This project was built to learn and implement real-world full-stack development concepts such as authentication, video handling, REST APIs, database management, and scalable MERN architecture.
